## 화면별 기능 상세

### 1. 로그인

| 기능ID | 1Depth | 기능분류 | 기능상세 | 서비스 목적 |
|--------|--------|---------|---------|-----------|
| CHATBOT-FU-01-001 | 로그인 | 관리자 로그인 | 관리자 로그인 | 운영자 권한 통제 |

**기능 설명 (정정):**
- 관리자 계정(ID/비밀번호) 입력 후 인증 요청
- 인증 성공 시 관리자 권한 검증 후 관리자 화면으로 이동
- 인증 실패 시 오류 메시지 표시, 로그인 상태에서만 관리자 메뉴 접근 가능

**서비스 상세:**
- 관리자 전용 기능 접근을 위해 계정 인증과 세션 확인이 필요
- 로그인 상태에 따라 관리자 메뉴 접근 권한을 분리하고 비인가 접근을 차단
- 상담 데이터, 지식 데이터, 개인정보 처리 기능을 보호하기 위한 기본 보안 절차

| CHATBOT-FU-07-002 | 내장 PII 패턴 | 활성/비활성(토글버튼) | 토글 클릭 시 해당 내장 패턴의 활성 상태를 즉시 변경·반영 |

| CHATBOT-FU-07-004 | 커스텀 패턴 | 활성/비활성(토글버튼) | 토글 클릭 시 해당 커스텀 패턴의 활성 상태를 즉시 변경·반영 |

| CHATBOT-FU-07-006 | 커스텀 패턴 | 수정(버튼) | 수정 버튼 클릭 시 선택 행 정보를 편집 폼에 표시, 저장 시 변경 내용으로 목록 갱신 |
| CHATBOT-FU-07-007 | 커스텀 패턴 | 삭제(버튼) | 삭제 버튼 클릭 시 confirm 창 표시, 확인 시 삭제 후 목록 갱신 |

**서비스 상세:**
- 개인정보 보호를 위해 기본 제공 PII 탐지 기준을 관리
- 내장 PII 패턴은 전화번호·이메일·주소 등 일반적 개인정보 유형 탐지 기준으로 사용
- 활성/비활성 제어로 업무 상황에 맞게 탐지 범위를 조정하고 오탐을 감소

**UI 컴포넌트:**
- 아코디언 섹션: 내장 패턴, 커스텀 패턴
- 각 항목: 토글 스위치 + 패턴명 + 정규식 설명
- 커스텀 패턴 추가 폼: 패턴 이름, 정규식(Regex), 문맥 단어 입력
- 수정/삭제: 행 우측 아이콘 버튼

### 참조: 실제 구현 화면 통합 구조 (보조 정보 — IA 기준 아님)

> 아래는 이미지에서 확인된 실제 화면 구성 참고용이다. 기능정의서 IA를 대체하지 않는다.

- **PII 관리** 화면: 단일 화면에 탭 4종으로 통합 (`패턴 상세 설정` / `스캔` / `스캔 이력` / `PII 테스트`)
  - `패턴 상세 설정` 탭 → 기능정의서의 **PII 패턴 관리(FU-07)** + **데이터 전처리 관리(FU-08)** 통합
  - `스캔` 탭 → 기능정의서의 **일괄 처리(FU-09-006~020)**
  - `PII 테스트` 탭 → 기능정의서의 **익명화 처리 단건 검출(FU-09-001~005)**
- 용어 대응: "복합어 분리 접미사" = NER 분리 접미사(FU-08-010~014), "지점명 감지·지역명 목록" = NER 지역명(FU-08-015~018), "품질 분류 옵션" = KMS 분류(FU-08-019~021)
